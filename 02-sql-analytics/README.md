# SQL Analytics Portfolio

## 📋 Overview
This folder contains SQL queries demonstrating advanced analytics skills for business intelligence and data analysis.

## 📊 Projects Included

### 1. Cohort Analysis (cohort_analysis.sql)

- Monthly cohort analysis
- Window functions and date manipulation

### 2. RFM Customer Segmentation (fm_analysis.sql)
- Customer value segmentation
- NTILE window functions
- Business logic with CASE statements

### 3. Funnel Analysis (unnel_analysis.sql)
- Conversion rate tracking
- Marketing funnel optimization
- Conditional aggregation

### 4. Window Functions (window_functions.sql)
- Rolling averages
- LAG/LEAD for trend analysis
- Ranking and time series

## 🛠️ Skills Demonstrated
- Common Table Expressions (CTEs)
- Window Functions (RANK, NTILE, LAG, LEAD)
- Date/Time manipulation
- Conditional aggregation
- Business metric calculation
- Query optimization

## 📈 Business Impact
These queries help businesses understand:
- Customer retention patterns
- High-value customer segments
- Marketing funnel efficiency
- Sales trends and patterns

These queries are designed for PostgreSQL but can be adapted for:
- Google BigQuery
- Amazon Redshift
- Snowflake
- MySQL (with slight modifications)
