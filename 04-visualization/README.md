# Sales Dashboard Portfolio

## 📊 Overview
This project contains interactive sales dashboards created with Tableau and Power BI, demonstrating data visualization and business intelligence skills.

## 📁 Files
- data/sales_data.csv - Main dataset (5,000+ transactions)
- data/daily_summary.csv - Daily aggregated data
- data/monthly_category.csv - Monthly sales by category
- data/regional_summary.csv - Regional performance
- data/product_performance.csv - Top products analysis
- data/customer_segments.csv - Customer segment analysis
- 	ableau/tableau_guide.txt - Tableau instructions
- powerbi/powerbi_guide.txt - Power BI instructions

## 📈 Dashboard Features
- **KPIs**: Total Sales, Profit, Orders, Average Order Value
- **Trends**: Monthly/Quarterly sales trends
- **Geography**: Regional sales performance
- **Products**: Top/bottom performing products
- **Customers**: Customer segmentation analysis

## 🛠️ Skills Demonstrated
- Data connection and preparation
- Calculated fields and measures
- Interactive filters and parameters
- Dashboard design best practices
